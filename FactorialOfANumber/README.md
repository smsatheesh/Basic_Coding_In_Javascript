# Factorial of a Number

Factorial of a Number is a simple JavaScript program (implemented in src/main.js) that calculates the factorial of a given non-negative integer using both iterative and recursive approaches.

## Features

- Takes user input for a non-negative integer
- Validates that the input is a valid non-negative integer
- Calculates factorial using an iterative approach (do-while loop)
- Calculates factorial using a recursive approach (arrow function)
- Displays both results to the user

## How to Run

1. Ensure you have Node.js installed.
2. Run the script:

```bash
npm install readline
node src/main.js
```

3. Enter a non-negative integer when prompted.

## Example

```
Factorial Calculator
Enter a non-negative integer: 5
Factorial (using iteration): 120
Factorial (using recursion): 120
```

```
Factorial Calculator
Enter a non-negative integer: 0
Factorial (using iteration): 1
Factorial (using recursion): 1
```

## Requirements

- Node.js (v12 or higher)
