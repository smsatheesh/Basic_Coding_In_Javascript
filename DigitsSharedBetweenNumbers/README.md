# Digits Shared Between Two Numbers

Digits Shared Between Two Numbers is a simple JavaScript program (implemented in src/main.js) that finds the common digits between two given numbers.

## Features

- Takes user input for two numbers
- Extracts the digits from each number
- Finds the digits that appear in both numbers
- Prints the shared digits or a message if none are found

## How to Run

1. Ensure you have Node.js installed.
2. Run the script:

```bash
npm install readline
node src/main.js
```

3. Enter the first number when prompted.
4. Enter the second number when prompted.

## Example

```
Enter the first number: 12345
Enter the second number: 54321
Shared digits between 12345 and 54321: 1, 2, 3, 4, 5
```

```
Enter the first number: 112233
Enter the second number: 445566
No shared digits between 112233 and 445566.
```

## Requirements

- Node.js (v12 or higher)
