# Temperature Conversion

Temperature Conversion is a simple JavaScript program (implemented in src/main.js) that converts temperatures between Celsius and Fahrenheit.

## Features

- Allows conversion from Celsius to Fahrenheit
- Allows conversion from Fahrenheit to Celsius
- Takes user input for the conversion option and temperature value
- Validates user input for both option and temperature
- Displays the converted temperature with two decimal places

## How to Run

1. Ensure you have Node.js installed.
2. Run the script:

```bash
npm install readline
node src/main.js
```

3. Select an option (1 or 2) when prompted.
4. Enter the temperature to convert when prompted.
