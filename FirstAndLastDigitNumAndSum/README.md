# First and Last Digit of Number and Its Sum

First and Last Digit of Number and Its Sum is a simple JavaScript program (implemented in src/main.js) that takes a non-negative integer from the user, computes the first and last digit of the number and their sum, and displays the result. It also handles single-digit numbers and invalid input.

## Features

- Takes user input for a non-negative integer
- Validates the input
- Computes the first and last digit of the number and their sum
- Handles single-digit numbers appropriately
- Displays the result

## How to Run

1. Ensure you have Node.js installed.
2. Run the script:

```bash
npm install readline
node src/main.js
```

3. Enter a non-negative integer when prompted.

## Example

```
First and Last Digit of Number and Its Sum
Enter a non-negative integer: 25
The first digit is 2 and last digit is 5 and sum of it's: 7
```

```
First and Last Digit of Number and Its Sum
Enter a non-negative integer: 5
The enter digit is single digit: 5
```

```
First and Last Digit of Number and Its Sum
Enter a non-negative integer: -5
Please enter a valid non-negative integer
```

## Requirements

- Node.js (v12 or higher)
