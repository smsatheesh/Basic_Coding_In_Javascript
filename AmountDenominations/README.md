# Amount Denominations

Amount Denominations is a simple JavaScript program (implemented in main.js) that breaks down a given amount into currency denominations (₹1000, ₹500, ₹100, ₹50, ₹20, ₹10, ₹5, ₹2, ₹1).

## Features

- Takes user input for an amount
- Calculates the number of each denomination needed
- Prints a clear breakdown of the denominations
- Shows the total amount

## How to Run

1. Ensure you have Node.js installed.
2. Run the script:

```bash
node main.js
```

3. Enter an amount when prompted.

## Example

```
Enter amount: 1867

Denomination Breakdown:
------------------------
1000: 1 x 1000 = 1000
500:  1 x 500  = 500
100:  3 x 100  = 300
50:   1 x 50   = 50
20:   0 x 20   = 0
10:   1 x 10   = 10
5:    1 x 5    = 5
2:    1 x 2    = 2
1:    0 x 1    = 0
------------------------
TOTAL: 1867
------------------------
```

## Requirements

- Node.js (v12 or higher)
