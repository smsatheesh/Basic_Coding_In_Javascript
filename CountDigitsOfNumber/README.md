# Count Digits of Number

Count Digits of Number is a simple JavaScript program (implemented in src/main.js) that counts the number of digits in a given non-negative integer.

## Features

- Takes user input for a number
- Validates that the input is a non-negative integer
- Calculates and displays the number of digits in the input number

## How to Run

1. Ensure you have Node.js installed.
2. Run the script:

```bash
npm install readline
node src/main.js
```

3. Enter a number when prompted.

## Example

```
Enter a number to count digits: 12345
Number of digits in 12345 is: 5
```

```
Enter a number to count digits: 0
Number of digits in 0 is: 1
```

## Requirements

- Node.js (v12 or higher)
